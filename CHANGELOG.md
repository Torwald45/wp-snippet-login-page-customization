# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/).

## [1.0.0] - 2025-11-04

### Added
- Custom logo support via WordPress Media Library attachment ID
- Changed logo URL from wordpress.org to site homepage using home_url()
- Custom logo title/tooltip for branding
- Unique prefixes (torwald45_logpage_) for all function names to prevent conflicts
- Lightweight implementation with no database queries
- Easy configuration - only two values to customize (logo ID and site name)
